## projek kalkulator sederhana Bunga dan Halena

## Deskripsi
Project ini merupakan aplikasi kalkulator sederhana berbasis **HTML, CSS, dan JavaScript**  
yang dapat melakukan operasi perhitungan dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian.  
Kalkulator kami juga menggunakan fitur tambahan seperti : 
- theme mode
- History

## Tujuan
- Melatih penggunaan HTML sebagai struktur halaman.
- Mengimplementasikan JavaScript untuk logika perhitungan.
- Memahami interaksi antara HTML dan JavaScript.
- Mengimplementasikan CSS ke HTML.

## Bahasa yang Digunakan
- HTML
- CSS
- JavaScript

## Framework yang digunakan
- Bootstrap

## Struktur File
- kalkulator.html
- script.js
- style.css

## Penjelasan File
- **index.html**  
  Berisi kerangka utama untuk tampilan yang ada di web kalkulator. Seperti Screen untuk layar, button angka, dan button simbol-simbol aritmatika untuk perhitungan. 

- **style.css**  
  Digunakan untuk mengatur tampilan dan tata letak halaman. Mengatur Warna, posisi, dan beberap hal terkait UI kalkulator.

- **script.js**  
  Berisi logika perhitungan dan interaksi tombol.
  Berisi setiap fungsi yang akan digunakan di kalkulator.

## Cara Menjalankan
1. Download atau clone repository
2. Download extension "Live Server"
2. Buka folder project
3. Klik kanan di file `index.html`
4. klik "open with live server"

## Cara Penggunaan
1. Klik tombol angka yang ingin di hitung
2. Pilih operator (+, −, ×, ÷)
3. Tekan tombol sama dengan (=)
4. Hasil perhitungan akan ditampilkan
5. Klik tombol clock untuk menampilkan halaman history, klik lagi untuk menutup halaman history.
6. klik tombol theme/mode untuk mengubah mode, klik lagi untuk mengubah kembali mode nya.
